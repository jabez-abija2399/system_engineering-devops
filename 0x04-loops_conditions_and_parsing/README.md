# 0x04. Loops, conditions and parsing

This directory contains Bash scripts demonstrating loops, conditional statements, and text parsing.

## Tasks

### 0. For Best School Loop
**File:** `1-for_best_school`

Bash script that displays "Best School" 10 times using a for loop.

### 1. While Best School Loop
**File:** `2-while_best_school`

Bash script that displays "Best School" 10 times using a while loop.

### 2. Until Best School Loop
**File:** `3-until_best_school`

Bash script that displays "Best School" 10 times using an until loop.

### 3. If 9, Say Hi!
**File:** `4-if_9_say_hi`

Bash script that displays "Best School" 10 times, but for the 9th iteration, displays "Best School" and then "Hi" on a new line.

### 4. 4 Bad Luck, 8 Is Your Chance
**File:** `5-4_bad_luck_8_is_your_chance`

Bash script that loops from 1 to 10 and displays "bad luck" for the 4th iteration, "good luck" for the 8th iteration, and "Best School" for other iterations.

### 5. Superstitious Numbers
**File:** `6-superstitious_numbers`

Bash script that displays numbers from 1 to 20 with special messages for 4, 9, and 17 using a case statement.

### 6. For ls
**File:** `8-for_ls`

Bash script that displays the content of the current directory in list format, showing only the part of the name after the first dash.

### 7. To file, or not to file
**File:** `9-to_file_or_not_to_file`

Bash script that gives information about the school file (existence, emptiness, and type).
