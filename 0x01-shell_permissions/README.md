# 0x01. Shell, permissions

This directory contains shell scripts for managing Linux file permissions and user operations.

## Tasks

### 0. My Name Is Betty
**File:** `0-iam_betty`

Script that switches the current user to the user betty.

### 1. Who Am I
**File:** `1-who_am_i`

Script that prints the effective username of the current user.

### 2. Groups
**File:** `2-groups`

Script that prints all the groups the current user is part of.

### 3. New Owner
**File:** `3-new_owner`

Script that changes the owner of the file hello to the user betty.

### 4. Empty!
**File:** `4-empty`

Script that creates an empty file called hello.

### 5. Execute
**File:** `5-execute`

Script that adds execute permission to the owner of the file hello.

### 6. James Bond
**File:** `8-James_Bond`

Script that sets the permissions of the file hello to -------rwx (no permissions for owner and group, all permissions for others).

### 7. John Doe
**File:** `9-John_Doe`

Script that sets the mode of the file hello to -rwxr-x-wx.

### 8. Directories
**File:** `11-directories_permissions`

Script that adds execute permission to all subdirectories for owner, group, and others (files are not changed).

### 9. More directories
**File:** `12-directory_permissions`

Script that creates a directory my_dir with permissions 751.

### 10. Change group
**File:** `13-change_group`

Script that changes the group owner of the file hello to school.
